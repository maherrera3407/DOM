El Modelo de Objeto de Documento (DOM)

El Modelo de Objeto de Documento (DOM) es una representación programática de la estructura de un documento HTML. Permite a los programadores acceder y manipular los elementos y contenidos de una página web a través del código JavaScript.

Por ejemplo, puedes cambiar el contenido de un elemento HTML, agregar nuevos elementos o eliminarlos, y modificar atributos de manera dinámica.

El DOM organiza los elementos de la página en un árbol de nodos, donde cada elemento es un nodo. Esto facilita la navegación y manipulación de la página mediante scripts.

En resumen, el DOM es una parte esencial para el desarrollo web interactivo y dinámico, ya que permite la creación de aplicaciones web complejas y ricas en funcionalidades.

Ejemplo de Acceso y Modificación del DOM:


// Accede a un elemento por su ID
var miParrafo = document.getElementById('miParrafo');

// Cambia el contenido de un párrafo
miParrafo.innerHTML = 'Este es un nuevo texto.';

En el ejemplo de código anterior, hemos accedido a un elemento con el ID "miParrafo" y hemos cambiado su contenido utilizando la propiedad `innerHTML` del elemento. Esto es solo un ejemplo simple de cómo puedes interactuar con el DOM.
